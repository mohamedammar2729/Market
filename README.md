#Market Website
This project is a fully responsive e-commerce website built using HTML, CSS, and ES6 JavaScript. It features a modern design and interactive functionality, allowing users to browse products, view a shopping cart, and register an account.

📌 Live Demo
You can view the live website here: Market Website

Features
🛒 E-Commerce Functionality
Display a variety of products categorized into Clothes, Books, and Electronics.
Add products to a shopping cart with automatic price calculation.
Display total price in the shopping cart view.
💻 Responsive Design
The website is fully responsive, using CSS Media Queries to ensure compatibility with all devices, including desktops, tablets, and mobile phones.
👤 User Account
A registration form where users can sign up with a name, email, and password.
🔗 Navigation
Smooth navigation between the home, about, and cart pages.
Logout functionality to simulate user session management.
🛠️ Technologies Used
HTML: Structured the content of the website.
CSS: Designed the layout, styling, and responsiveness.
Used media queries for responsiveness.
JavaScript (ES6): Added interactivity, including:
Dynamic product rendering.
Cart functionality (adding items, displaying total).
Button and navigation interactivity.
💡 Installation and Usage
To run this project locally, follow these steps:

Clone the repository:
git clone https://github.com/mohamedammar2729/Market.git

Navigate to the project folder:
cd Market
Open index.html in your preferred web browser.
